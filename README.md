---

```
- Clone esse repositório:
$ git clone https://github.com/valdir-alves3000/studywithfather
- Entre no diretório:
$ cd studywithfather
- Instale as dependências:
$ npm install
# or
yarn install
- Inicie o app:
$ npm run dev
# or
yarn dev
```
